tinyxml2 + tinyxml2-ex
======================

TinyXML2 is a simple, small, efficient, C++ XML parser that can be easily integrated into other programs.

TinyXML2-ex is a set of add-on classes and helper functions bringing basic XPath element selection and C++11/14 features, such as iterators, strings and exceptions, to tinyxml2.

The master is hosted on github:
https://github.com/leethomason/tinyxml2
https://github.com/stanthomas/tinyxml2-ex

The online HTML version of these docs:
http://leethomason.github.io/tinyxml2/
https://github.com/stanthomas/tinyxml2-ex/blob/master/USAGE.md
